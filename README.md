# Micro-Bench
一款专为 ESP32-C3 开发板设计的轻量级、全维度性能测试工具。

🚀 项目简介 (Introduction)

本项目是一个基于 MicroPython 的综合跑分程序，专门针对 ESP32-C3 (RISC-V) 核心进行适配。它不依赖任何第三方显示驱动库，直接通过原生 FrameBuffer 测试硬件极限性能。

🌟 核心功能 (Key Features)

• 硬件自动感知： 自动检测 CPU 频率、内存总量、固件版本及存储空间。

• 算力评估： 针对 RISC-V 架构的整数与浮点运算能力测试（质数算法 & 三角函数）。

• 内存带宽： 模拟大规模数据拷贝，测试 SRAM 的读写效率。

• 原生图形渲染： 绕过驱动层限制，测试 FrameBuffer 在 16bit 色深下的逻辑绘图 FPS。

• 量化评分系统： 综合各项指标，输出直观的“综合性能得分”。

🛠️ 如何使用 (Quick Start)

1. 确保你的 ESP32-C3 已烧录 MicroPython 固件。

2. 将 main.py（即跑分脚本）上传至开发板。

3. 在终端查看详细输出，或结合 Web 门户查看 UI 展示。

💯分数分析

• 均衡能手 (Score 150 - 300): 完美适配 Web 服务器、带有动画效果的 GUI 界面、物联网网关。

• 基础型 (Score < 150): 建议仅用于简单的传感器上报、深度休眠定时器任务。
